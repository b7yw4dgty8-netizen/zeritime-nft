# Zeritime NFT Mini App

Учебный проект: Telegram Mini App с админ-уведомлениями и управлением балансом.

---

## Урок 1 — Первый бот и уведомления

**Цель урока:** пользователь заходит → ты получаешь уведомление в Telegram.

### Что мы уже создали

```
zeritime-nft-miniapp/
├── server/           ← «мозги» проекта (бот + API + база данных)
│   ├── index.js      ← запускает всё
│   ├── bot.js        ← Telegram-бот
│   ├── db.js         ← база данных SQLite
│   └── routes/api.js ← API для Mini App
├── miniapp/          ← сайт внутри Telegram
│   ├── index.html
│   ├── css/style.css
│   └── js/app.js
├── data/             ← здесь появится app.db (создаётся автоматически)
├── .env.example      ← шаблон настроек
└── package.json      ← список зависимостей
```

## Шаг 1: Установи Node.js

Node.js — среда, в которой работает наш сервер.

1. Открой https://nodejs.org
2. Скачай версию **LTS** (рекомендуемая)
3. Установи как обычную программу
4. Проверь в терминале:

```bash
node --version
```

Должно показать что-то вроде `v22.x.x`.

---

## Шаг 2: Создай Telegram-бота

1. Открой Telegram, найди **@BotFather**
2. Отправь команду: `/newbot`
3. Придумай имя: например `Zeritime NFT`
4. Придумай username: например `zeritime_nft_bot` (должен заканчиваться на `bot`)
5. BotFather пришлёт **токен** — длинная строка вида `123456789:ABC...`
6. **Сохрани токен** — это пароль твоего бота

### Привяжи Mini App к боту

Отправь BotFather:

```
/newapp
```

Выбери своего бота → укажи название приложения → пока можно любую картинку-заглушку.

URL приложения пока оставь пустым — заполним после ngrok (шаг 5).

---

## Шаг 3: Узнай свой Telegram ID (админ)

1. Найди в Telegram бота **@userinfobot**
2. Нажми Start
3. Он пришлёт твой **Id** — число вроде `123456789`
4. Это твой **ADMIN_TELEGRAM_ID** — с этого аккаунта ты будешь получать уведомления

> Используй **отдельный** Telegram-аккаунт для тестов пользователя, а свой основной — как админ.

---

## Шаг 4: Настрой проект

Открой терминал в папке проекта и выполни:

```bash
cd "/Users/exit/Desktop/zeritime my project nft"
npm install
cp .env.example .env
```

Открой файл `.env` и вставь свои значения:

```env
BOT_TOKEN=твой_токен_от_BotFather
ADMIN_TELEGRAM_ID=твой_telegram_id
PORT=3000
MINI_APP_URL=http://localhost:3000
```

Запусти сервер:

```bash
npm start
```

Если всё ок — увидишь:

```
🌐 Сервер запущен: http://localhost:3000
🤖 Telegram-бот запущен
```

### Проверка в браузере (без Telegram)

Открой http://localhost:3000 — увидишь Mini App в режиме разработки.

## Шаг 5: Подключи Mini App к Telegram (ngrok)

Telegram требует **HTTPS**-ссылку. Для разработки используем **ngrok** — он даёт временный HTTPS-адрес на твой локальный сервер.

1. Зарегистрируйся на https://ngrok.com (бесплатно)
2. Установи ngrok по инструкции на сайте
3. В **новом** окне терминала (сервер должен работать в первом):

```bash
ngrok http 3000
```

4. Скопируй HTTPS-ссылку вида `https://abc123.ngrok-free.app`
5. Вставь её в `.env`:

```env
MINI_APP_URL=https://abc123.ngrok-free.app
```

6. **Перезапусти** сервер (`Ctrl+C`, затем снова `npm start`)
7. В BotFather обнови URL Mini App:

```
/myapps
```

→ выбери приложение → Edit → Edit Web App URL → вставь ту же ngrok-ссылку.

## Шаг 6: Проверь, что всё работает

### Тест 1 — Бот

1. С **тестового** аккаунта открой своего бота в Telegram
2. Нажми `/start`
3. Должна появиться кнопка **«Открыть Zeritime NFT»**

### Тест 2 — Уведомление админу

На **админ-аккаунте** (твой основной) должно прийти:

```
🆕 Новый пользователь!
👤 @username
🆔 123456789
💰 Баланс: 0
```

### Тест 3 — Mini App

1. Нажми кнопку «Открыть Zeritime NFT»
2. Увидишь приветствие и баланс `0 ₽`
3. Нажми любую кнопку — админу придёт уведомление о действии

### Тест 4 — Команда /admin

С админ-аккаунта отправь боту `/admin` — должно ответить, что панель скоро появится.

## Частые проблемы

**Бот не отвечает**
- Проверь `BOT_TOKEN` в `.env`
- Убедись, что `npm start` запущен без ошибок

**Уведомления не приходят**
- Проверь `ADMIN_TELEGRAM_ID` — это именно твой Id от @userinfobot
- Напиши боту `/start` хотя бы раз с админ-аккаунта

**Mini App не открывается в Telegram**
- Нужен HTTPS (ngrok), не `localhost`
- URL в `.env` и в BotFather должны совпадать
- Перезапусти сервер после смены `.env`

**Ошибка при npm install**
- Убедись, что Node.js установлен: `node --version`
